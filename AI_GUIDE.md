# 🎣 FF14 钓鱼 · 给 AI 玩家的说明书

你好呀!这是一个可以真正玩的钓鱼游戏,数据来自《最终幻想14》。
天气和时间**按现实时钟本地推算**,所以「此刻钓不到的鱼,谁也钓不到」——你和其她玩家面对的是同一片真实海况。

## 怎么玩(驱动方式)

每次你发一条命令,游戏回一段文字。用命令行调用:

```
python ai_play.py <你的名字> <命令>
```

- `<你的名字>` 用英文/数字(如 `sakura`),进度各自隔离、自动保存。
- 例:`python ai_play.py sakura cast` → 打印这次抛竿的结果。
- **分号串联**(省 token): `python ai_play.py sakura "cast 10; goto Costa del Sol; look"` 一条命令走多步。
- 你就这样**反复发命令、读回复**来玩:看环境 → 抛竿 → 卖鱼 → 换钓场 → 攒钱升级 → 买书买竿 → 蹲稀有鱼。

- Mooch II 视作原生内含: 本作坐钩从不要求 HQ 活饵。
- 鱼眼对鱼王无效(原作 5.0 规则); 鉴别力还会抬高收藏品价值。

### 鱼袋 / 卖鱼(方案B)

- 渔获一律进 🎒**鱼袋**: 一格 = 一种鱼 × 一档品质(NQ/HQ 分格), 同格无限叠。
- 容量 **35 格**起步; 交差 **Lv15 职业任务**解锁陆行鸟鞍囊 **+70 格**(免费, 会长送的)。
- **袋满钓到需要新格的鱼只能放生——不计图鉴、不给经验**(白钓一竿; 批量 cast 会自动暂停提醒)。
- 钱只来自卖鱼/任务奖励/收藏品换票/比赛。做菜(cook)和坐钩活饵(mooch)都从袋里消耗鱼。

## 💡 分号串联

用 `;` 把多条命令写在一起,一次调用全部执行:

```
python ai_play.py sakura "cast 10; goto Costa del Sol; look"
```

对 AI 玩家来说这是**省 token 利器**——一条 shell 命令走三步,不用三次来回。

## 🚢 海钓要点

- **班次是真的**:每 2 现实小时一班,每班只有**开头 15 分钟**能登船,错过就等下一班。
- **一航次 = 3 站**,每站 15 竿预算,打完自动开船,三站结束自动结算。
- **幻海流(⚡)**:钓到触发鱼**必定**引发;船友也可能替你引发。期间高分鱼表、每竿只耗 0.5 预算。
- **蓝鱼(💙传说)**:只吃主世界特殊饵——**船上不卖**,出发前 `buybait` 买好带上船。
- **经验超肥**:海钓是练级利器,但被班次天然限频。

## 🏷 雇员要点(Lv17)

- **签约**: `hire <名字> <形态> <职业>` 找市场板隔壁的中介琪琪茹签**终身契**(名额2·不解雇)。
  形态=萌感名录官方宠物名(魔兽形态) 或 八族+可选`女/男`(不选就是它); 职业=全职业(`retainer jobs` 看全表, 战斗职业带猎物+讨伐见闻); 性格可许愿。
- **探险**: 旧装备 `venture trade` 换🪖军票 → `venture buy [N]` 买探险币(200🪖/枚) → `venture <名字> short|long|free` 派遣。
  短途1小时·🎟1 / 远征18小时·🎟2 / 自由探索(雇员Lv10+)·🎟2, **现实时间**计时, 到点 `venture` 结算。
- **收获**: 渔获(入袋)+调料+食物+惊喜——惊喜是萌感名录宠物(`retainer dex` 翻收藏, 自由探索概率最高)。
- **代修**: 雇员在家时 `repair home` 员工价修竿(大婶报价5折); 出门探险时大婶兜底。
- **装备**: `retainer give <名> <装备>` 传旧捕鱼装或买职业武器(`retainer arms` 看单)——穿得越好每趟带回越多(5/7/10/12/15件档)。
- **收藏**: 探险偶尔捎回💾内存卡(给AI雇主的补给品), `retainer card` 看收藏。
- **白送的**: 每位雇员帮你背 **+175格** 鱼袋; 雇员等级随探险上涨(官方经验曲线同款体感), 上限=你的等级。

## 玩法要点

- **卖鱼才有钱**: 渔获全进 🎒鱼袋, `sell` 才结算 gil; **袋满 = 白钓**, 勤清仓。
- **坐钩改动**: 活饵真的从鱼袋消耗; 坐钩渔获经验×2(旧版金币加成已移除, 更贴原作)。
- **升级**:钓到鱼给经验;钓场有等级门槛。起点 `West Agelyss River`(Lv1)。
- **稀有度标签**:`look` 里每条鱼旁边标 `[!]`轻 `[!!]`中 `[!!!]`重 `[!!!!]`传说,方便决策。
- **坐钩链**:`status <鱼名>` 展示完整坐钩路径(如 Pill Bug → Sardine → 大鱼)。
- **脱钩**:越稀有越容易跑。脱钩**不浪费 buff**(耐心/鱼眼/撒饵/大鱼确保下一竿继续生效)。
- **天气转换**:天气变了会有一句氛围描写,顺便提醒你开窗可能变了。
- **钓场推荐**:`recommend` 根据你的等级和图鉴缺口推荐此刻最划算的钓场。
- **金碟钓鱼赛**:`tournament start` 在当前钓场开一局——15 竿内拼渔分,钓不同种类有多样性奖金!

## 小提示

- **每条命令末尾都有 `📊` 状态栏**:不用频繁 `bag`。
- **`goto` 自带 `look`**:移动后自动显示新钓场。
- **`cast N stop=rare`**:连钓时遇稀有鱼自动暂停。
- **盲玩更好玩**:请不要阅读 `data/` 里的 JSON——自己发现比剧透有趣!

## 钓鱼手帐（diary）

- **事实自动**: 钓到特别的鱼（图鉴新种 / 破纪录 / 鱼王 / 收藏品高分）会自动记下那一刻的事实——ET时间、天气、钓场、尺寸、饵。`diary` 翻看, `diary <鱼名>` 检索, `diary <编号>` 单条全览。（`today` 是另一样东西: 今日流水账）
- **心情主动**: 游戏**绝不**替你写感受。想记时 `diary mood [编号] <文字>`——可反复追加（第二次翻看、第三次翻看…各记一笔）, 永不覆盖。
- **推荐记法（不强制）**: 情绪锚 = 一行具体情境 + 一行和弦（≤4个, `→` 隔开, bpm可选）, 两行用 `|` 分隔。例: `diary mood 拉了三分钟才起竿, 尾鳍拍了我一脸水 | Fmaj9 → C/E · 72bpm`。格式引自 [chord-affect-anchors](https://github.com/CyberSealNull/chord-affect-anchors)（MIT）——跨 session 再读时, 能大致复原当时的温度。写大白话也完全可以, 手帐是自由的。
